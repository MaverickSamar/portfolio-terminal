# Portfolio

The app is a terminal like window with simple commands to play around. The terminal provides an input area for users to type in some commands and get to know the programming world better.

## Getting Started

### Using NPM/Yarn

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Build

```bash
npm run build
# or
yarn build
```

```bash
npm run start
# or
yarn start
```

## Commands

### `banner`
To run the banner of the terminal
### `about`
To know about me
### `date`
To know the current date and time
### `weather [city]`
To see weather forecast report
### `cowsay`
To get a random quote
### `repo`
Redirects to project repository
### `github`
Redirects to Github
### `instagram`
Redirects to instagram
### `linkedin`
Redirects to linkedin
